# 🔐 Diffie-Hellman Key Exchange

A simple implementation of the **Diffie-Hellman Key Exchange algorithm**, demonstrating how two parties can securely establish a shared secret over an insecure communication channel.

---

## 📌 Overview

This project focuses on the core concept of secure key exchange using public-key cryptography. It highlights how secure communication can be achieved without directly transmitting sensitive information.

---

## 🚀 Features

* Clean and minimal implementation
* Easy to understand and modify
* Suitable for beginners in cryptography
* Demonstrates secure key exchange concept

---

## 🛠️ Technologies Used

* Concepts: Modular arithmetic, cryptography basics
